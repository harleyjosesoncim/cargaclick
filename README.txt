# CargaClick - ZIP de Integração Tela Simulação de Frete

Este pacote inclui:

- config/routes.rb (rotas com collection :calcular)
- app/controllers/fretes_controller.rb (controller completo com create e calcular)
- app/views/fretes/calcular.html.erb (formulário visual, campos completos e botão de localização)

## Como usar:
1. Faça backup dos arquivos originais.
2. Extraia na raiz do projeto Rails e aceite substituir os arquivos.
3. Rode rails server.
4. Acesse http://localhost:3000/fretes/calcular

Se aparecer erro de banco/modelo, me avise!
